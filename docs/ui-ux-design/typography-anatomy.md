# Typography Anatomy -- Ultimate Cheat Sheet

> **All important typography terms and definitions for graphic and product designers in one place.**

---

## Type Positioning & Spacing

### Vertical Reference Lines

| Line                | Description                                                                              |
| ------------------- | ---------------------------------------------------------------------------------------- |
| **Ascender Height** | The top of the tallest lowercase letters (b, d, f, h, k, l, t)                           |
| **Cap Height**      | The top of uppercase letters (A, B, C, etc.)                                             |
| **X-Height**        | The height of lowercase letters without ascenders or descenders (a, c, e, m, n, o, etc.) |
| **Baseline**        | The invisible line where all letters sit                                                 |
| **Descender Line**  | The bottom of letters that extend below the baseline (g, j, p, q, y)                     |

### Horizontal Spacing

| Term                          | Description                                                                |
| ----------------------------- | -------------------------------------------------------------------------- |
| **Letter Spacing / Tracking** | The uniform spacing between all characters in a block of text              |
| **Kerning**                   | The adjustment of space between two specific characters for visual balance |
| **Leading / Line-Spacing**    | The vertical distance between lines of text (baseline to baseline)         |

## Type Anatomy

### Structural Parts

| Term          | Description                                                          | Example Letter |
| ------------- | -------------------------------------------------------------------- | -------------- |
| **Ascender**  | The part of a lowercase letter that extends above the x-height       | f              |
| **Descender** | The part of a letter that extends below the baseline                 | q              |
| **Stroke**    | A straight or curved line that forms the main body of a letter       | A              |
| **Stem**      | The main vertical stroke of a letter                                 | B              |
| **Leg**       | A lower stroke attached to the stem at one end and free at the other | k              |
| **Arm**       | An upper horizontal or diagonal stroke attached at one end only      | k              |
| **Ear**       | A small decorative stroke extending from the top of lowercase g      | g              |
| **Shoulder**  | A curved stroke originating from a stem                              | n              |
| **Tail**      | A descending stroke, often decorative                                | Q              |
| **Spine**     | The main curved stroke of the letter S                               | S              |
| **Swash**     | A decorative extension or stroke on a letter                         | A (italic)     |
| **Bar**       | A horizontal stroke across the middle of a letter                    | f              |

### Serifs & Terminals

| Term              | Description                                                   | Example Letter |
| ----------------- | ------------------------------------------------------------- | -------------- |
| **Serif**         | A small line or stroke attached to the end of a larger stroke | n              |
| **Terminal**      | The end of a stroke that doesn't have a serif                 | t              |
| **Ball terminal** | A circular form at the end of a stroke                        | r              |

### Enclosed & Open Spaces

| Term             | Description                                | Example Letter |
| ---------------- | ------------------------------------------ | -------------- |
| **Bowl**         | The curved, enclosed part of a letter      | D              |
| **Counter**      | The fully enclosed space within a letter   | O              |
| **Open-counter** | A partially enclosed space within a letter | y              |

### Connections & Decorative Elements

| Term            | Description                                                           | Example Letter |
| --------------- | --------------------------------------------------------------------- | -------------- |
| **Link**        | The stroke connecting the upper and lower bowls of a double-story g   | g              |
| **Loop**        | The enclosed lower part of a double-story g                           | g              |
| **Arc of stem** | A curved stroke flowing into or out of a stem                         | j              |
| **Apex/Vertex** | The point where two strokes meet at the top (apex) or bottom (vertex) | M              |
| **Crotch**      | The interior angle where two strokes meet                             | w              |
| **Joint**       | The point where a stroke connects to a stem                           | L              |
| **Gadzook**     | A decorative flourish connecting two letters                          | ct             |
| **Ligature**    | Two or more letters combined into a single glyph                      | ae             |

## Practical Tips

- Adjust leading and tracking to preserve the good legibility of your content.
- Make sure that kerning in your logo designs is well-balanced.
- Use consistent type scales across your design system to maintain visual hierarchy.
- Test readability at different sizes -- what works at 16px may not work at 12px.
