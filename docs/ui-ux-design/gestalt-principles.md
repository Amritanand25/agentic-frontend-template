# 6 Gestalt Principles in UX Design

> **Visual grouping principles have been invented by Gestalt psychologists and are based on human's innate ability to perceive objects as organized patterns and simplify complex compositions.**

---

## The 6 Principles

### 1. Good Figure

Objects grouped together tend to be perceived as a single figure. Tendency to simplify.

**In UI:** Users perceive a card with an icon, title, and description as one unit rather than three separate elements. Design components as cohesive wholes.

---

### 2. Proximity

Objects tend to be grouped together if they are close to each other.

**In UI:** Form labels placed close to their inputs are perceived as belonging together. Spacing between form groups signals separate sections. Related actions (Save/Cancel buttons) should be close together.

---

### 3. Similarity

Objects tend to be grouped together if they are similar.

**In UI:** All primary action buttons share the same color and shape, so users recognize them as the same type of element. Consistent styling signals consistent function.

---

### 4. Continuation

When there is an intersection between two or more objects, people tend to perceive each object as a single uninterrupted object.

**In UI:** Users follow a horizontal navigation bar as a continuous strip even when items are separated. Breadcrumbs, progress bars, and timelines leverage continuation.

---

### 5. Closure

Visual connection or continuity between sets of elements which do not actually touch each other in a composition.

**In UI:** Users can perceive incomplete shapes as complete -- a circle icon with a gap is still read as a circle. Loading spinners, progress indicators, and partially visible cards use closure to suggest completeness.

---

### 6. Symmetry

Objects tend to be perceived as symmetrical shapes that form around their center.

**In UI:** Symmetrical layouts feel balanced and stable. Two-column layouts with equal cards, centered modals, and mirrored navigation patterns leverage symmetry for visual harmony.

---

## Practical Tips

- Organize UI and content elements based on Gestalt principles to reduce user's cognitive load and improve perception.
- Use the Proximity law to position logically connected elements together.
- Use the Similarity principle to achieve consistency in UI or composition.
