# HEART Framework -- UX Metrics Map

> **HEART is a framework of user-centric metrics developed by Google to measure success of design and help product and development teams make informed decisions that address business and user needs.**

HEART framework from Google is a helpful tool for measuring the success of design solutions. It helps set up a user-centric approach on a team and classify UX metrics into 5 categories: Happiness, Engagement, Adoption, Retention and Task Success. The framework is widely used by many product teams and companies and allows them to leverage quantitative and qualitative user data in the design process.

---

## The Framework

| Category         | Goals                  | Signals                                              | Metrics                                                 |
| ---------------- | ---------------------- | ---------------------------------------------------- | ------------------------------------------------------- |
| **H**appiness    | User satisfaction      | Users feedback from surveys, interviews              | Satisfaction rating, net promoter score                 |
| **E**ngagement   | User content discovery | The amount of time users spend in the app            | Number of shares, average sessions length, page views   |
| **A**doption     | User onboarding        | App downloads, new registrations, using new features | Download rate, registration rate, feature adoption rate |
| **R**etention    | User loyalty           | Returning users, subscription renewals               | Subscription renewal rate, churn rate                   |
| **T**ask success | User goals completion  | Usability studies, user behaviour analytics          | Task completion                                         |

### Column Definitions

- **Goals** -- Critical user tasks that define what success looks like
- **Signals** -- Channels of getting UX metrics (how you observe user behavior)
- **Metrics** -- Quantifiable UX metrics (what you actually measure)

---

## Practical Tips

- Define UX metrics for every project. It's important to measure the results of design efforts and understand how it impacts user experience and enhances the product's value.
- HEART framework is convenient for measuring large projects at scale but also its principles can be implemented for smaller tasks.
- Make sure all your stakeholders are on board with the established metrics and you have a shared understanding of what you are measuring against.
